# Spam Email Classifier

## Objective
Detect whether an email is Spam or Not Spam.

## Technologies Used
- Python
- Pandas
- Scikit-Learn

## Algorithm
Naive Bayes Classifier

## Features
- Converts email text into numerical features
- Trains a machine learning model
- Predicts whether an email is spam

## Future Improvements
- Larger email dataset
- Web interface
- Real-time email filtering
